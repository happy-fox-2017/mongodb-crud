>`npm install`

the routing

|    ROUTING    |  METHOD  |    DESCRIPTION      |
|:--------------|:--------:|:--------------------|
|/api/     |GET       | Get All data Books  |
|/api/:id |GET       | Get one data Book   |
|/api/    |POST      | Input data Book     |
|/api/:id |PUT       | Update data Book    |
|/api/:id |DELETE    | Delete data Book    |
